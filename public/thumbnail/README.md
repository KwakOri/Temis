# 썸네일 이미지 폴더

이 폴더에는 템플릿별 썸네일 이미지가 저장됩니다.

## 파일 명명 규칙

- 파일명: `[template_id].png`
- 예시: `12345678-1234-1234-1234-123456789abc.png`

## 사용법

1. 관리자 페이지에서 템플릿 생성
2. 생성된 템플릿의 ID 확인
3. 이 폴더에 `[template_id].png` 형식으로 썸네일 이미지 추가
4. 자동으로 템플릿 목록에 썸네일이 표시됨

## 권장 사양

- 형식: PNG (권장)
- 크기: 400x300px 또는 16:9 비율
- 파일 크기: 1MB 이하
